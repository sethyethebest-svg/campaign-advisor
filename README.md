# Campaign Advisor

Public static web demo for a weather-aware marketing campaign recommender.

The site is designed to be deployed directly from the repository root with GitHub Pages.

Current product story:

- Scenario-led bubble tea operator use cases
- Live weather forecast demo with P-ROAS/CAC ranking
- Transparent model and data-source explanation
- 20-chain U.S. bubble tea benchmark universe
- Future iOS companion app teaser
